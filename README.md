## Prerequisites

- go >=1.16.6
- azure-cli >= 2.22.1
- fzf >= 0.20.0

## Install

### Brew (Linux/Mac)

```sh
brew tap riweston/aztx
brew install aztx
```

### Scoop (Windows)

```sh
brew tap riweston/aztx
brew install aztx
```

### Manual

Grab a release from the repo and install it.

### Build from source

```sh
go install github.com/riweston/aztx
```
